# Ia_yaroslav
Носко Ярослав Юрьевич
Training Planner — это простое приложение для учёта и планирования тренировок с удобным графическим интерфейсом. Оно позволяет добавлять тренировки, указывая дату, тип и длительность, а также фильтровать их по этим параметрам. Все данные сохраняются в файл JSON и автоматически загружаются при запуске, при этом приложение проверяет корректность ввода: дата должна быть в верном формате, а длительность — положительным числом.
